一、项目介绍
1) 项目类型
- 基于 uni-app + Vue3 + uview-plus 的多端前端项目（支持 H5、App 构建脚本）。
- 项目主题是“AI 学习助手/数字老师”，包含聊天、数字人形象、成长档案、账号与登录等功能模块。

2) 技术结构
- 前端核心目录：src/pages（页面）、src/services（接口封装）、src/main.js、src/pages.json。
- 接口调用统一走 src/services/http.js + src/services/api.js。
- 登录态使用本地存储 token（src/services/storage.js）。
- 用户基础资料使用本地存储（src/services/userProfile.js）。
- 项目内含大量后端接口文档（backend-apis、docs），用于前后端联调。

3) 部署与联调特征
- Vite 代理与 API 源都指向同一后端域名：
	https://jkyhobdhqqah.sealoshzh.site
- H5 开发态通过 /api 代理，生产/非 H5 走线上域名。
